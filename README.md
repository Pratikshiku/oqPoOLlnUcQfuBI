## Teacher Workload Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/0bc1b387806545e3890bfb37c8c6eb10/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Teacher User, Administrator.

**Teacher User Functions:**
1. Login
2. Register
3. Personal Center (Change Password, Personal Information)
4. Clock-in Management
5. Announcement Information Management
6. Salary Management
7. Workload Management
8. Notification Information Management


**Administrator Functions:**
1. Login
2. Homepage
3. Personal Center (Change Password, Personal Information)
4. Clock-in Management
5. Basic Data Management (Announcement Type Management, Workload Type Management, Notification Type Management)
6. Announcement Information Management
7. Salary Management
8. Workload Management
9. Notification Information Management
10. Teacher Management
